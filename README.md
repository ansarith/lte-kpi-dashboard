# 📊 LTE KPI Dashboard

This project provides an interactive **Streamlit dashboard** for monitoring LTE network KPIs.

The dashboard allows users to visualize performance indicators, filter by site or cell, and export reports as **PNG charts or PowerPoint presentations**.

---

## 🚀 Features

- KPI visualization using Plotly charts
- Filter by **LNBTS (site)** and **LNCEL (cell)**
- Daily aggregation option
- Site-level KPI grouping
- Export charts to **PNG**
- Export KPI report to **PowerPoint**
- Interactive dashboard using **Streamlit**
